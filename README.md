# X-analysis Project

## Install Backend Dependencies:  
```bash  
cd backend  
npm install  
```  

## Run Backend:  
```bash  
cd backend  
npm start  
```  

## Install Frontend Dependencies:  
```bash  
cd frontend  
npm install  
```  

## Run Frontend:  
```bash  
cd frontend  
npm run dev  
```
